# Emoji Guessing Game

This project is a 2-emoji guessing game with a Node.js backend and a Godot frontend (frontend to be developed). The backend serves emoji combinations and hints, while the frontend will provide the user interface for the game.

## Project Structure

-   `/backend`: Contains the Node.js Express server.
-   `/frontend`: (Planned) Will contain the Godot game client.

## Backend

The backend is a Node.js Express server that provides API endpoints for:
-   Generating two-emoji combinations with correct answers and distractor options.
-   Providing hints for a given phrase (emoji combo or answer).

It uses the Google Gemini API for content generation.

For detailed setup and API documentation, see the [backend/README.md](backend/README.md).
